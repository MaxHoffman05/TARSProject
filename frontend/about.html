About the Project

Interstellar is my favorite movie and I wanted to combine it with coding. In this project I wanted to better learn how to interact and build API's. I wanted to learn how to better prompt AI to generate better responses, and I believe I got pretty close to how TARS would act.

In theory this project could be turned into a chatbot for any character that you can think of, you just need to modify the prompts in the code.

If you are curious about the code behind this project you can find it here. It goes into more detail about the project and how to get it running.

I learned...

  • FastAPI
  • Redis in Python
  • How to interact with Groq's API
  • Creating better prompts for LLM's
  • More Bootstrap knowledge
  • More JavaScript knowledge
